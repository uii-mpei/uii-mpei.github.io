Инструкция по установке ПО

Это краткая версия, как установить такую же среду, как в лаборатории. Больше вариантов (например, для Mac или не CodeBlocks) — в общей инструкции.

В Windows XP каталоги установки и проектов не должны содержать пробелов и кириллицы.

Компилятор GCC 8.1 (MinGW-w64)

Создать каталог C:\MinGW и распаковать в него архив.

Официальный сайт GCC: https://gcc.gnu.org.
Официальный сайт MinGW-w64 (версия для Windows): https://mingw-w64.org.

Среда разработки CodeBlocks

  1. Установить CodeBlocks, все по умолчанию, но в конце не запускать среду.
  2. Открыть «Проводник», в строку адреса ввести %AppData%\Roaming и нажать Enter (для редактирования адреса надо нажать на значок папки слева от адреса).
  3. Распаковать в открывшуюся папку архив настроек. (Если каталог CodeBlocks уже существует, заменить его.)

Официальный сайт: http://www.codeblocks.org.

Система контроля версий Git

Просто установить.

Официальный сайт: http://git-scm.com.

Сборочная система CMake

При установке выбрать вариант «Add CMake to the system PATH for all users».

Официальный сайт: https://cmake.org.
